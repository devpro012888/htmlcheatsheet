HyperText Markup Language (HTML) Cheatsheet

What is HTML?

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

Paragraphs <p>

A paragrpah consists of one or more sentences that form a self-contained unit of discourse. The start of a paragraph is indicated by a new line.

To create a paragraph, surround the words that make up the paragraphwith an opening <p> tag and closing </p> tag.
Bold <b> This is how we make a word appear bold By enclosing words in the tags <b> and </b> we can make characters appear bold.
Italic <i> This is how we make a word appear italic By enclosing words in the tags <i> and </i> we can make characters appear in italic.
Superscript <sup> On the 4th of September you will learn about E=MC2. The <sup> element is used to contain characters that should be superscript such as the suffixes of dates or mathematical concepts like raising a number to a power such as 22.
Subscript <sub> On the 4th of September you will learn about E=MC2. The amount of CO2 in the atmosphere grew by 2ppm in 20091. The <sub> element is used to contain characters that should be subscript. It is commonly used with foot notes or chemical formulas such as H2O.
Line Breaks <br />

The Earth
gets one hundred tons heavier every day
due to falling space dust.

 As you have already seen, the browser will automatically show each new paragraph or heading on a new line. But if you wanted to add a line break inside the middle of a paragraph you can use the line break tag.
Horizontal Rules <hr />

Venus is the only planet that rotates clockwise.


Jupiter is bigger than all the other planets combined.

 To create a break between themes, such as a change of topic in a book or a new scene in a play - you can add a horizontal rule between sections using the <hr /> tag.
Strong <Strong>

Beware: Pickpockets operate in this area

This toy has many small pieces and is not suitable for children under five years old.

 The use of the <Strong> element indicates that its content has strong importance. For example, the words contained in this element might be said with strong emphasis.
Emphasis <em>

I think Ivy was the first.

I think Ivy was the first

I think Ivy was the first

 The <em>element indicates emphasis that subtly changes the meaning of a sentence.
Quotations <blockquote>
<q>

Did you ever stop to think, and forget to start again

As A.A. Milne said, Some people talk to animals. Not many listen though That's the problem.

The <blockquote> element is used for longer quotes that take up an entire paragraph.

The <q> element is used for shorter quotes that site within a paragraph.

Abbreviations and Acronyms <abbr>
<acronym>

Prof Stephen Hawking is a theoretical physicist and cosmologist

NASA do some crazy space stuff.

 If you use an abbreviation or an acronym, then use <abbr> element can be used. A title attribute on the opening tag is used to specify the full term.
Citations <cite>

A Brief History of Time by Stephen Hawkiing has sold over ten million copies worldwide

 When you are referencing a piece of work such as a book, film or research paper, the <cite>element can be used to indicate where the citation is from.
Definition <dfn>

A black holeis a region of space from which nothing, not even light, can escape.

 The <dfn> element is used to indicate the defining instance of a new term.
Address <address>

homer@example.org

742 Evergreen Terrace, Springfield.

The <address> element has quite a specific use: to contain contact details for the author of the page

It can contain a physical address, but it does not have to. For example, it may also contain a phone number or email.

Insert and Delete <ins>
<del>

It was the worst best idea she had ever had

 The <inst> element can be used to show content that has been inserted into a document, while the <del> element can show text that has been deleted from it.
Strike Through <s>

Laptop computer

Was $995

Now only $375

 The <s> element indicates something that is no longer accurate or relevant (but that should not be deleted).
