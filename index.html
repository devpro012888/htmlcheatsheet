What is HTML?

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

Paragraphs <p>

A paragrpah consists of one or more sentences that form a self-contained unit of discourse. The start of a paragraph is indicated by a new line.

To create a paragraph, surround the words that make up the paragraphwith an opening <p> tag and closing </p> tag.
Bold <b> This is how we make a word appear bold By enclosing words in the tags <b> and </b> we can make characters appear bold.
Italic <i> This is how we make a word appear italic By enclosing words in the tags <i> and </i> we can make characters appear in italic.
Superscript <sup> On the 4th of September you will learn about E=MC2. The <sup> element is used to contain characters that should be superscript such as the suffixes of dates or mathematical concepts like raising a number to a power such as 22.
Subscript <sub> On the 4th of September you will learn about E=MC2. The amount of CO2 in the atmosphere grew by 2ppm in 20091. The <sub> element is used to contain characters that should be subscript. It is commonly used with foot notes or chemical formulas such as H2O.
Line Breaks <br />

The Earth
gets one hundred tons heavier every day
due to falling space dust.

 As you have already seen, the browser will automatically show each new paragraph or heading on a new line. But if you wanted to add a line break inside the middle of a paragraph you can use the line break tag.
Horizontal Rules <hr />

Venus is the only planet that rotates clockwise.


Jupiter is bigger than all the other planets combined.

 To create a break between themes, such as a change of topic in a book or a new scene in a play - you can add a horizontal rule between sections using the <hr /> tag.
